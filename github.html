Langkah Pembuatan GitHub Page Menggunakan Sistem Git

1. Membuat repository baru diakun GitHub masing - masing (atau melalui )

2. Beri nama repository tersebut dengan format "usernama.github.io"

3. Setelah itu klik "create" dibagian bawah.

4. Selanjutnya buka folder yang berisikan tugas web, setelah klik kanan, dan pilih "Git Bash Here"

5. Setelah beberapa saat akan muncul GitBash. Lalu ketikan "git init" untuk mengecek directory git saat ini.

6. Lalu ketikan "git clone http://alamat-clone-.." untuk proses cloning ke repository.

7. Selanjutnya, pindahkan file dalam folder yang akan diclone ke folder (dalam hal ini adalah) ary-uts.github.io

8. Lalu ketikan "git add ." umtuk me-add folder baru ke repository GitHub.

9. Setelah beberapa saat, ketikan "git push -u origin main" untuk me-push folder ke repository GitHub.

10. Untuk mengecek berhasil atau tidaknya proses "clone", bukalah repository github tujuan atau dengan me-refresh halamannya.
